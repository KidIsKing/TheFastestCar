# TheFastestCar - Игра на Pygame

### **Бесконечная езда по шоссе с видом сверху.**

![Фото с демкой игры](images/TheFastetsCar_demo.jpg)

## Описание

- Суть игры заключается управлении машиной на бесконечном шоссе в две полосы в одну сторону, две в другую.
- Главной задачей является максимально долгое время выживания на дороге для заработка наибольшего количества очков или денег ради покупки новых автомобилей (планируется добавить целый автопарк, возможно, с улучшениями).
- Врагами будут являться машины, едущие равномерно в едином направлении с игроком или обратном с разной скоростью.
- Важнейшую роль в оценке игры играет использование сложных или не очень алгоритмов в коде. Планируемое применение их:
    1.  сложное появление врагов;
    2.  постепенное увеличение скорости игрока (то есть игрок двигается с ускорением - const?) и врагов, которые с каждой минутой (или другим заданным значением) будут появляться с всё более большей скоростью;
    3.  модель повреждения и отнимаемое количество здоровья при столкновении;
    4.  случайная генерация бонусов на дороге (аля Fastlane);
    5.  определение меры улучшения при прокачке авто игрока (опционально, зависит от добавления автопарка).

## Используемые технологии

- python 3.10.11
- PyGame 2.6.1

## Запуск

1\. Создать виртуальное окружение:

```bash
python -m venv venv
```

2\. Активировать его:

```bash
source venv/Scripts/activate
```

3\. Установить зависимости:

```bash
pip install -r requirements.txt
```

4\. Обновить версию `pip` (опционально):

```bash
python.exe -m pip install --upgrade pip
```
